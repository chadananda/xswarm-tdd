# Domain: plan-api — API Product Planning

End-to-end API product planning: idea → viability → strategy → technical → marketing.
Use `/plan-api` skill. Four linear phases, each produces a deliverable document.
Tough-love advisor: one question at a time, back challenges with research data, be specific with numbers.
Score ≥10 on viability (5 factors × 0-3, multiplicative) to proceed.
Developer-focused: personas over demographics, DX over UX, usage-based pricing, SDK-first distribution.
Reference files in `skills/plan-api/references/` — load per-phase, not all at once.
